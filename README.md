# Biz2.0

This repository documents the legal, economic, and technical architecture
for a constitutionally governed company that separates:

- Ownership (Equity)
- Value Allocation
- Incentives and Labor Rewards

The system is designed to:
- Start as a U.S. LLC
- Operate with directive-based value allocation
- Use token-like systems for tracking and enforcement
- Convert cleanly to a corporation if desired
- Remain compliant with U.S. securities, tax, and labor law

This is not a DAO.
This is not a cryptocurrency startup.
This is a legally grounded organization with programmable economics.
